安装

Redis 官网：https://redis.io/

Redis 在线测试：http://try.redis.io/

Window

下载地址：https://github.com/microsoftarchive/redis/releases

方式一、自动安装

msi 文件下载完成双击安装即可。

方式二、源码安装

源码下载完成解压文件到 C 盘，重命名为 redis，打开 cmd 窗口，切换到目录 C:\redis 运行：

建议添加 redis 路径到环境变量方便操作。

另打开一个 cmd 窗口，切换到 redis 目录，原窗口不可关闭，不让无法访问服务端。

Linux

方式一、源码安装

下载地址：https://redis.io/download

Ubuntu

方式一、自动安装

PHP-Redis

下载地址：https://github.com/phpredis/phpredis/releases

安装文档：https://github.com/phpredis/phpredis/blob/develop/INSTALL.markdown

Window

下载地址：https://pecl.php.net/package/redis

  1. 查看 phpinfo 信息，获取 PHP 版本信息，线程安全信息：ts 、nts，结构：x86 、x64
  2. 下载对应版本扩展，解压压缩包，复制 php_redis.dll 至对应 PHP 版本扩展文件夹内，重启服务器即可。

Linux

  1. 安装 PHP Redis 扩展
  1. 配置PHP

方式一：在 /etc/php/7.2/mods-available 目录下创建 redis.ini 文件，添加 extension=redis.so，并创建对应配置文件软链接

方式二：直接修改配置文件 php.ini ，添加 extension=redis.so

  1. 重启服务器

发布订阅

Redis 发布订阅(pub/sub)是一种消息通信模式：发送者(pub)发送消息，订阅者(sub)接收消息。

Redis 客户端可以订阅任意数量的频道。

性能测试

redis 性能测试工具可选参数如下所示：

序号 选项 描述 默认值
1 -h 指定服务器主机名 127.0.0.1
2 -p 指定服务器端口 6379
3 -s 指定服务器 socket
4 -c 指定并发连接数 50
5 -n 指定请求数 10000
6 -d 以字节的形式指定 SET/GET 值的数据大小 2
7 -k 1=keep alive 0=reconnect 1
8 -r SET/GET/INCR 使用随机 key, SADD 使用随机值
9 -P 通过管道传输 请求 1
10 -q 强制退出 redis。仅显示 query/sec 值
11 –csv 以 CSV 格式输出
12 -l 生成循环，永久执行测试
13 -t 仅运行以逗号分隔的测试命令列表。
14 -I Idle 模式。仅打开 N 个 idle 连接并等待。
